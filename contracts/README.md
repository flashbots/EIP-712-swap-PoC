# contracts

EIP712 contract(s).

Built with [Forge](https://github.com/foundry-rs/foundry/tree/master/forge).

## install dependencies

```sh
./scripts/install.sh
```

## build contract(s)

```sh
./scripts/build.sh
```

## deploy contract(s) to chain

```sh
./scripts/deploy.sh
```

## run tests

```sh
forge test --chain-id 9001 --revert-strings debug -vvv
```
